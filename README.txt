Data Visualization Tool
======================

This application provides data visualization, preprocessing, and machine learning capabilities 
with an easy-to-use graphical user interface.

Installation
-----------
To install the required dependencies, run:

python -m pip install -r requirements.txt

Usage
-----
To start the application, run:

python main.py

Features
--------
- Data loading from CSV and ARFF files
- Data preprocessing (cleaning, normalization, handling missing values)
- Data visualization (box plots, QQ plots, scatter plots)
- Classification models (KNN, Naive Bayes, Decision Tree, Neural Network)
- Regression models (Linear Regression, Neural Network)
- Model evaluation and prediction
